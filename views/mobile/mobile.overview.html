Appverse Mobile

Appverse Mobile is an open-source mobile development platform developed by GFT Appverse. It enables developers to use HTML5, JavaScript, and CSS3 and leverage native functionalities of mobile devices to build apps tailored for those devices. The platform currently supports Android, iOS, and Windows and provides a broad set of APIs to access that native functionalities via a consistent and uniform JavaScript interface for the supported operating systems.

Why Appverse Mobile?

Appverse Mobile brings you all the benefits of GFT's 6+ years of development in a corporate setting, where time-to-market, robustness, and security are critically important. The platform has been open-sourced so that the community can benefit from this work and indeed contribute to make it even better! You don't need to waste time on the technical details of individual devices, and you have the freedom to use native device functionality from your HTML5 app. So you can focus on your next cutting-edge app while we make sure it runs optimally on Appverse Mobile.

Depth vs Breadth

Appverse Mobile provides deep integration with native device functionality by giving developers a rich, fine-grained JavaScript API with which to enable their HTML5 apps. Our JavaScript APIs are consistent across platforms, so you can focus on your app and not the technical specifications of the device – we keep the platform consistent for you!
